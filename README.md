# Exploratory Data Analysis (EDA) Project

This project performs automated Exploratory Data Analysis (EDA) on an Excel dataset.  
It includes data cleaning, summary statistics, visualization, and statistical hypothesis testing.

The project is structured like a real-world data analysis pipeline with modular design.

---

## 🚀 Features

- ✅ Automatic Excel file loading
- ✅ Missing value handling (Median & Mode strategy)
- ✅ Auto column detection (GPA, Participation, Preference, etc.)
- ✅ Summary statistics
- ✅ Grouped analysis
- ✅ Data visualizations
- ✅ Correlation heatmap
- ✅ ANOVA test
- ✅ Chi-Square test
- ✅ Cleaned dataset export
- ✅ Excel summary export
